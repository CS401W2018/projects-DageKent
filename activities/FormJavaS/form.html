What is JavaScript? How did it get its name?

A programing laguage, created for first browser. Netscape developed it, was called LiveScript, back in the days Java was famous, they renamed it to attract more attention, called it JavaScript.

What are the two ways you can run JavaScript?

One runs in the browser, JavaScript code can direct embedded into HTML documents, and call from browsers, such using script tags. Also, JavaScript can run on servers, using such as Node.js environment, can call JavaScript from servers.

What was the most popular front-end js library?

JQuery was the most popular front-end js library, it simpified DOM manipulating, event processing, make develop to be more convinient.

What is the most popular front-end js library today?

React.js is the most popular front-end js library today, developed by Facebook, based on component architecture, more popular use in modern Web development.
