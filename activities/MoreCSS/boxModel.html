What is the difference between block tags and inline tags?

Block tags occupy full width available, and start in a new line. Inline tags only occupy the width of their content, and don't start a new line.

What is the margin, border, and padding?

Margin is the space outside an element that separate it from other element. Border is the edges surround an element. Padding is the inner space of element, between content and border.

What are units of measurement you can use in CSS?

There are some easurement units can use in CSS, such as px, cm, mm, in, pt, %, em, rem, etc

When would you use a figure tag?

Figure tag is used to wrap media elements, such as pictures, diagrams, also add figcaption tag to give explanation.
