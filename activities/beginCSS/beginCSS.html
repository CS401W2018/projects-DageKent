Explain the difference between inline, embedded, and external CSS.

Inline CSS refers to the direct use the style attribute inside HTML tags to define styles. Such as changing colors or fonts. An embedded CSS refers to style tag used inside the head tag section of a HTML file. external CSS refers to write CSS code for define style in an independent CSS file, and use link tag in HTML to connect CSS file. Explain why you should only use external CSS.

Inline CSS used more for small range revising. Embedded CSS apply on single page or quick testing more. external CSS keeps HTML organized and clean, easier maintenance and updates. external CSS allows browsers cache styles, which helps reload website faster. external CSS also good for use in multiple pages repeatedly.So, external CSS is the best option
