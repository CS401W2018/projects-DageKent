Why do we need a viewport meta tag?

It ensures that the website can scaling Correctly in different devices. For example, ensuring website can scale and fit screen size correctly on mobile devices, provides better user experience.

When you select the same tag twice, and in one of them you say "flex-direction: column;" and the other says "flex-direction: row;", which one takes priority?

Later CSS code will cover earlier code. For example, using "flex-direction:column" first, then type "flex-direction:row". The "row" will replace "column".

How do you change the CSS for a web page displayed on a phone?

Using media queries, such as @media ONLY screen AND (max-width: 700px). The px in parentheses indicates the content present will change when screen width reach the specific value.
