Why don't we use the border attribute?

The border attribute is an outdated HTML attribute. Today, web design uses CSS to control and style tables, CSS makes code more flexible and easier to maintain.

What is the easiest way to color every other row a different color?

Using the CSS nth-child selector, such as tr:nth-child(even) { background-color: blue; }. This will make even-numbered rows automatically change colors, avoid the need to manually set each row's color.
