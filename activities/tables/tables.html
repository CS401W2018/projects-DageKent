When do you use a table?

When need to present the structured data, such as timetable, transcripts, and financial statement.

When do you not use a table?

When need to create a page layout or organize simple content, should use CSS not a table.

When do you need to do extra work for table accessibility?

When a screen reader needs to recognize the table, should add elements such as caption and th tags to improve accessibility.
