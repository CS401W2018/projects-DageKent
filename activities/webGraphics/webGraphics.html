What kind of photos are good to encode using JPEG?

JPEG is good for real life pictures, such as landscapes and portraits.

What kind of photos are good to encode using GIF?

GIF is good for images with few colors, such as cartoons and animations

What is unique about the PNG format?

PNG is unique for lossless compression, alpha channel, or transparency. It works well for cartoons and logo images. It also has a wider color range than GIF.

How do you resize your photo to the size it should be displayed?

There are different ways to resize a photo, such as using Microsoft Paint software on a Windows computer to resize photo. Or Typing code in a CSS file or HTML file to set the width and height in pixels to resize a photo.

Why should you resize photos using photo editing software instead of resizing it using CSS?

Photo editing software can reduce file size and increase web page loading speed. Photo editing software also can retain quality of images, because use CSS to resize images may make the image blurry. An image resized by photo editing software can make CPU calculation faster, because the image is already in the right size, doesn't need additional process.
