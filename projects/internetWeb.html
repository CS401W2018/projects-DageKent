Internet

A global network system that uses information sharing and communication

Web

An Internet-based that provides access to websites and hyperlinks. It uses HTTP protocol

How are the Internet and Web different?

The Internet is the foundation of the network, and the Web is a service run on the Internet. Web used for information display and access.

ASCII and Unicode

ASCII is the US standard for English, and Unicode is the unified standard that supports all languages

What is UTF-8

A Unicode encoding method, uses 1 to 4 bytes equate to characters. Mainly used for web pages and documents

URL

Navigate internet resources, like web pages, and domain names.

Domain Name

Internet adress, like google.com. It is used to replace IP addresses.

HTTP

A protocol used to transmit data between web clients and servers.

HTTPS

A secure version of HTTP, transmission of data is encrypted.

IP Address

The unique ID of a device in the network.

DNS

It converts domain names into IP addresses, helping users to access websites more easily.

W3C Recommendations

The World Wide Web Consortium publishes standards. It ensures standardization.

XML, XHTML, HTML, HTML5

XML is a language to save and transmit data, structured but with no restrictions to data content. XHTML is a combined HTML and XML, a stricter version of HTML. HTML is a language used to establish website structures and content. HTML5 is the latest version of HTML, it is good for modern web development.
