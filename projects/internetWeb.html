Internet

A global network system that uses information sharing and communication via protocols such as TCP/IP. It is fundamental to all online services.

Web

A service based on the Internet. It uses HTTP protocol to access websites and hyperlinks to obtain and display data.

How are the Internet and Web different?

The Internet is the foundation of the network, and the Web is a service run on the Internet. Web used for information display and access.

ASCII and Unicode

ASCII is a character encoding standard of American for English and Basic symbols, and Unicode is the unified standard of character encoding that supports all languages.

What is UTF-8

A Unicode encoding method. Mainly used for web pages and documents. It can work with ASCII.

URL

Navigate and tag internet resources positions, like web pages, and domain names.

Domain Name

Internet resource address, like google.com. It is used to replace IP addresses.

HTTP

A protocol transmits data between web clients and servers. It is not a secure protocol.

HTTPS

A secure version of HTTP, transmission of data is encrypted.

IP Address

It is like a unique ID of a device in the network. There are IPv4(32 bits) and IPv6(128 bits)

DNS

It converts domain names into IP addresses, helping users to access websites more easily.

W3C Recommendations

The World Wide Web Consortium publishes standards. It ensures web standardization and openness.

XML, XHTML, HTML, HTML5

XML is a language to save and transmit data, with no restrictions to data content. XHTML combines HTML and XML, a stricter version of HTML. It has stricter rules. HTML is a language used to establish website structures and content. HTML5 is the latest version of HTML. It has many new functions. One of the new functions is inputting audio and video directly without extension.
